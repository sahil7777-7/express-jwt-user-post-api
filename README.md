# Node.js Authentication & Post API

A REST API built with Node.js, Express.js, MongoDB, Mongoose, JWT Authentication and Cookies.

## Features

### User

- User Registration
- User Login
- User Logout
- Update User Profile
- Search User

### Posts

- Create Post
- Update Post
- Delete Post
- Search Posts

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Cookie Parser
- dotenv

## API Endpoints

### User

POST /register

POST /login

POST /logout

PUT /updateuser

GET /searchuser

### Posts

POST /createpost

PUT /updatepost

DELETE /deletepost

GET /searchpost
